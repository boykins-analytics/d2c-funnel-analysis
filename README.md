# AI-Driven D2C Funnel Analysis

## Problem
Identify where users drop off between first click and checkout in a D2C shop.

## Approach
- Funnel analysis using structured data
- KPI-based evaluation of conversion steps
- Dashboard-driven insights
- AI-assisted interpretation of results

## Key Metrics
- Conversion rate
- Drop-off rate per funnel stage
- Checkout friction indicators

## Outcome
This project demonstrates how data and AI tools can be used to quickly identify conversion blockers and propose actionable improvements.
